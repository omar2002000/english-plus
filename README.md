# 📚 English Plus — تطبيق إدارة الحصص والمراكز التعليمية (الإصدار 3.0)

تطبيق ويب احترافي (PWA) لإدارة الطلاب والمجموعات والحضور والمدفوعات والتقارير ومركز واتساب — **يعمل أوفلاين بالكامل** بدون إنترنت.

## 🆕 ما الجديد في الإصدار 3.0؟

### 🔒 1. أمان PIN متقدم
- **قفل تلقائي بعد 5 محاولات فاشلة** (5 دقائق)
- **إجبار تغيير الـ PIN الافتراضي** (1234) عند أول تفعيل
- عداد المحاولات المتبقية يظهر للمستخدم
- شاشة قفل واضحة مع العدّاد التنازلي

### 📱 2. يعمل أوفلاين 100%
- كل البيانات محلية (IndexedDB)
- لا يحتاج أي إنترنت للعمل
- تمت إزالة قسم المزامنة السحابية المشفّرة
- استُبدل بـ **نسخ احتياطي يدوي واضح** (JSON)

### 📅 3. نسخ احتياطي تلقائي أسبوعي
- كل أسبوع يُصدّر التطبيق نسخة احتياطية تلقائياً
- تُحفظ في مجلد التنزيلات (Downloads)
- يمكن إيقافه/تفعيله من الإعدادات
- يعرض تاريخ آخر نسخة تلقائية

### 🚀 4. بدء فارغ (لا بيانات وهمية)
- التطبيق يبدأ فارغاً تماماً عند أول تشغيل
- رسالة ترحيب واضحة مع 3 خيارات:
  - إضافة طالب
  - إنشاء مجموعة
  - أو إضافة بيانات تجريبية (اختياري)
- يمكن إضافة البيانات التجريبية يدوياً من الإعدادات

### 🔐 5. روابط ولي الأمر آمنة
- **انتهاء صلاحية اختياري** (أسبوع/شهر/3 شهور/سنة/دائم)
- **الحد الأقصى لعدد الفتح** (10/50/100/غير محدود)
- عدّاد فتح لكل رابط
- إلغاء الروابط بسهولة
- عرض جميع الروابط النشطة

### 🏆 6. شهادات تلقائية قابلة للطباعة
- **3 أنواع شهادات**:
  - 🏆 شهادة تفوق (للحاصلين على 36+/40)
  - ✅ شهادة التزام (للحضور 90%+)
  - 📈 شهادة تحسّن (للتحسّن الملحوظ)
- تصميم احترافي "English Plus" بألوان ذهبية
- تتضمن: اسم الطالب، الصف، المجموعة، المتوسط، نسبة الحضور، المركز (إن كان من أوائل الطلبة)
- تُولّد كـ PDF جاهز للطباعة أو الإرسال

### 🧹 7. تنظيف package.json
- إزالة الـ scaffold غير المستخدم
- تمت إزالة: Prisma، @mdxeditor، @dnd-kit، react-hook-form، next-auth، next-intl، next-themes، react-syntax-highlighter، sharp، uuid، zod، z-ai-web-dev-sdk، وغيرها
- تم الإبقاء فقط على المكتبات المستخدمة فعلياً
- تقليل حجم node_modules بنسبة ~60%

### 💳 8. نظام المدفوعات الشامل
- منع الدفع الزائد (يُعدّل تلقائياً للمتبقي)
- 3 أوضاع: كامل، جزئي، مبلغ مخصص
- **دفع جماعي**: اختر مجموعة → طلاب → مبلغ → دفع
- **حذف دفعات متعددة**: بالتحديد ثم حذف
- 4 تبويبات: آخر الشهر، أول الشهر، دفع جماعي، السجل

### 📷 9. إصلاح الماسح الضوئي
- فحص دعم المتصفح للكاميرا أولاً
- طلب الإذن صراحة مع رسائل خطأ مفصلة
- محاولة الكاميرا الخلفية ثم أي كاميرا كـ fallback
- إعادة المحاولة مرتين قبل إظهار الخطأ

### 📅 10. مواعيد متعددة للمجموعة
- إضافة عدة مواعيد للمجموعة (يوم + ساعة + فترة)
- عرض جميع المواعيد في الجدول
- حذف مواعيد منفردة

### 🔔 11. شاشة الإشعارات
- زر الإشعارات يفتح شاشة مخصصة
- تفعيل إشعارات المتصفح
- فلاتر (الكل/غير مقروء/مقروء)
- تعليم الكل كمقروء + مسح الكل

### 🔙 12. زر رجوع سفلي
- زر رجوع عائم في كل الشاشات
- يسهّل التنقل على الموبايل

### 💬 13. مركز واتساب الجديد (35 قالب)
- 35 قالب جاهز في 9 تصنيفات
- **ربط بالبيانات الحقيقية** للطالب
- معاينة مباشرة بعد استبدال المتغيرات
- تنبيه البيانات الناقصة قبل الإرسال
- بحث + تصنيف + تصفية
- إرسال فردي + جماعي + سجل

## ✨ المميزات الكاملة

### 🏠 لوحة التحكم
- مؤشرات يومية، إيرادات، متأخرات، أفضل الطلاب
- **توصيات ذكية يومية** (غائبو آخر حصة، متأخرو السداد، المتفوقون)
- إجراءات سريعة

### 👨‍🎓 إدارة الطلاب
- بيانات كاملة + صورة + QR/Barcode + PIN 4 أرقام
- ملف كامل (4 تبويبات: بيانات، حضور، مدفوعات، بطاقة)
- **مدير الباقات والخصومات** (6 أنواع باقات + فحص أهلية تلقائي)
- **توليد شهادات PDF** (تفوق/التزام/تحسّن)
- **روابط ولي الأمر آمنة** (مع صلاحية وحد فتح)
- بحث وفلترة واستيراد/تصدير Excel
- أرشفة بدل الحذف

### 👥 إدارة المجموعات
- **مواعيد متعددة** للمجموعة الواحدة
- جدول كامل، نظام دفع، حساب الدخل المتوقع
- ربط الطلاب، سجل الحصص

### 📷 الحضور والغياب
- 4 طرق: يدوي، QR/Barcode، PIN، استيراد
- المسح المتدفق + الماسح الذكي + وضع البوابة (Kiosk)
- العداد الحي + الإغلاق الذكي + إرسال غياب واتساب
- **AR Cards** (عرض بيانات الطالب فوق البطاقة عند المسح)

### 📝 حصة اليوم والتقييم
- جدول سريع (حضور/حفظ/مراجعة/واجب من 40)
- فلاتر + ترتيب + تنبيهات للمتأخرين
- تقرير يومي PDF + إرسال واتساب

### 💳 الاشتراكات والمدفوعات
- منع الدفع الزائد (تعديل تلقائي)
- **دفع جماعي** + **حذف دفعات متعددة**
- فواتير PDF تلقائية + تذكير جماعي

### 📊 التقارير (12 نوع)

## 🚀 طريقة التشغيل

- Node.js 18+ أو Bun

### التثبيت
```bash
# فك الضغط
unzip english-plus-app-v3.zip
cd english-plus

# تثبيت التبعيات
```

## 🎯 خطوات الاستخدام السريع

1. **أول تشغيل**: التطبيق فارغ — ابدأ بإضافة مجموعة وطالب
2. **من الإعدادات**: 
   - فعّل القفل (سيُطلب تغيير PIN 1234)
   - فعّل النسخ الاحتياطي التلقائي الأسبوعي
   - فعّل الإشعارات الذكية + التوصيات + التحكم الصوتي (اختياري)
3. **أضف طلابك** (أو استورد من Excel)
4. **أنشئ مجموعاتك** بمواعيد متعددة واربط الطلاب
5. **سجّل الحضور** بالباركود أو PIN
6. **قيّم الحصة** واحفظ
7. **سجّل المدفوعات** (دفع كامل/جزئي/جماعي)
8. **صدّر الشهادات** الشهرية للمتفوقين
9. **شارك روابط أولياء الأمور** (مع صلاحية وحد فتح)
10. **خذ نسخة احتياطية** يدوياً أو دع التطبيق يفعلها أسبوعياً
